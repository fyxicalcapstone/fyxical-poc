# fyxical-poc
Proof of concept for rule-based clinical decision support in physical therapy

## Overview
Rule-based + ML decision support system for outpatient orthopedic PTs.
Built on Google Vertex AI + BigQuery.

## Tech Stack
- Google Vertex AI Workbench
- BigQuery (fyxical-poc)
- Python, scikit-learn, Pandas

## Notebooks
1. Data Loading
2. Exploratory Data Analysis
3. Decision Tree MVP
4. Logistic Regression
5. Results Export
